# Familiar dependencies — everything needed to run, in one file.
# Install:  py -m pip install -r requirements.txt            (Windows)
#           python3 -m pip install -r requirements.txt       (macOS / Linux)
#
# Tip: a brand-new Python release (e.g. 3.14 the week it ships) may not have
# prebuilt wheels for every compiled package yet, which makes pip try to build
# from source and fail. If you hit that, use Python 3.11–3.13.
